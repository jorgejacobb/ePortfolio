# Capstone Project

## EcoGrid Renewable Energy Site Selection Platform

My capstone project focuses on the development of a full stack web application designed to help users evaluate and compare potential renewable energy locations. The platform will simulate how software engineering, databases, mapping systems, and secure coding practices can work together to support decision-making in clean energy technology.

The application will allow users to:

* Explore renewable energy site locations
* Visualize locations using interactive mapping tools
* Compare environmental and energy-related data
* Rank locations using recommendation logic
* Access a secure administrative dashboard
* Interact with a responsive and user-friendly interface

### Planned Technologies

* Angular
* Node.js
* Express.js
* MongoDB
* REST APIs
* JWT Authentication
* Mapping Integration
* Secure Coding Practices

### Software Engineering Focus

This project is intended to demonstrate:

* Full stack development skills
* Secure coding implementation
* Database integration and management
* Software design and architecture
* User-centered interface design
* API development
* Authentication and route protection

### Future Vision

In the future, I would like to continue expanding this project by integrating more advanced renewable energy analysis tools and interactive geographic systems. My goal is to combine software engineering with sustainability-focused technologies and create applications that can contribute to smarter environmental decision-making.
